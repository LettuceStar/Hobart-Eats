# Hobart Eats
A small end-to-end web application built with **ASP.NET Core MVC** that simulates a real-world restaurant review platform.
Users can register and log in to browse restaurants and submit ratings and reviews, while admins manage restaurants, menu items, and customer support tickets through a protected back-office interface.

---

## Tech Stack
- **C# / ASP.NET Core MVC**
- **Entity Framework Core**
- **SQLite** (local development & demo)
- **ASP.NET Core Identity** (authentication & roles)
- **Python** (data analysis & reporting scripts)
